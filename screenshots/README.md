# Screenshots

This directory contains screenshots of Lilypad in action.

## Current Screenshots

All screenshots are automatically generated using Playwright:

- `dashboard-light.png` - Light mode dashboard
- `dashboard-dark.png` - Dark mode dashboard
- `bulk-actions.png` - Bulk actions feature

## How to Regenerate

Screenshots are automatically generated by Playwright. To regenerate:

```bash
# Ensure Lilypad is running
docker run -d -p 8080:8888 -v /var/run/docker.sock:/var/run/docker.sock ghcr.io/gaving/lilypad:latest

# Run Playwright tests
npx playwright test screenshots.spec.ts
```

## Screenshot Specifications

- **Resolution:** 1200 x 800 pixels
- **Browser:** Chromium
- **Format:** PNG

## Playwright Configuration

See `playwright.config.ts` for test configuration.
See `screenshots.spec.ts` for screenshot test definitions.
