# Screenshots

This directory contains screenshots of Lilypad in action.

## Desktop Screenshots

All desktop screenshots are 1200x800 pixels:

- `dashboard-light.png` - Light mode dashboard with running containers
- `dashboard-dark.png` - Dark mode dashboard with running containers  
- `bulk-actions.png` - Bulk actions feature with checkboxes selected

## Responsive Screenshots

- `dashboard-mobile.png` - Mobile view (375x812 - iPhone size)
- `dashboard-tablet.png` - Tablet view (768x1024 - iPad size)

## Regenerating Screenshots

Screenshots are automatically generated by Playwright. To regenerate:

```bash
# Ensure Lilypad is running with containers
docker run -d -p 8080:8888 -v /var/run/docker.sock:/var/run/docker.sock ghcr.io/gaving/lilypad:latest

# Run Playwright tests
npx playwright test screenshots.spec.ts
```

## Playwright Configuration

- **Desktop:** 1200x800 (Chrome)
- **Mobile:** 375x812 (iPhone X)
- **Tablet:** 768x1024 (iPad)

See `screenshots.spec.ts` for test definitions.
